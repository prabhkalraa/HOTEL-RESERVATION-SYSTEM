# Hotel Reservation System

A Java-based application that simulates a hotel booking system, allowing users to search rooms, make bookings, cancel reservations, and manage check-outs. The system uses Object-Oriented Programming principles and supports file handling for data persistence.

## Features

- Search available rooms by category (Standard, Deluxe, Suite)  
- Book rooms with basic payment validation  
- Cancel bookings and update room availability  
- Check-out functionality to free occupied rooms  
- View all booking records  
- Persistent storage using file handling  

## Tech Stack

- Java  
- Object-Oriented Programming (OOP)  
- Collections Framework (ArrayList)  
- File Handling (BufferedReader, BufferedWriter)

## How to Run

1. Clone or download the repository  
2. Open the project in any Java IDE (IntelliJ, Eclipse, or VS Code)  
3. Compile and run `Main.java`  
4. Follow the menu options to interact with the system
